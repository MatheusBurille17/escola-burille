# Imagem do Sistema de Graduação

## Localização
Adicione a imagem do sistema de graduação do Jiu-Jitsu em:
```
/public/images/sistema-graduacao-jiu-jitsu.jpg
```

## Especificações Recomendadas
- **Formato**: JPG ou PNG
- **Dimensões**: 800x1000px (ou proporção similar vertical)
- **Tamanho**: Máximo 500KB
- **Conteúdo**: Tabela/infográfico mostrando as faixas do sistema kids/juvenil

## Como Adicionar
1. Salve sua imagem com o nome: `sistema-graduacao-jiu-jitsu.jpg`
2. Coloque na pasta: `public/images/`
3. A imagem aparecerá automaticamente na página de Jiu-Jitsu

## Dica
A imagem deve mostrar claramente:
- As cores das faixas (branca, cinza, amarela, laranja, verde)
- Faixas intermediárias (com combinações de cores)
- Idades correspondentes
- Número de aulas por grau

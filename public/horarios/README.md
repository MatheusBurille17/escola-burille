# Horários - PDFs das Modalidades

Esta pasta contém os PDFs com os horários completos de cada modalidade.

## Arquivos necessários:

- `jiu-jitsu.pdf` - Horários completos do Jiu-Jitsu (adulto e infantil)
- `judo.pdf` - Horários completos do Judô
- `taekwondo.pdf` - Horários completos do Taekwondo
- `move-fight.pdf` - Horários completos do Move & Fight

## Como adicionar os PDFs:

1. Coloque os arquivos PDF nesta pasta com os nomes exatos listados acima
2. Os PDFs serão abertos em uma nova aba quando o usuário clicar no card da modalidade no popup
3. Certifique-se de que os nomes dos arquivos estejam em minúsculas e sem espaços

## Formato recomendado para os PDFs:

- Tamanho: A4
- Orientação: Retrato ou Paisagem (conforme necessário)
- Conteúdo sugerido:
  - Logo da escola
  - Nome da modalidade
  - Tabela completa de horários
  - Informações sobre professores
  - Observações importantes
  - Contato para dúvidas
